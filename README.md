# xlsxTotxt
convert xlsx to txt
</br>
How to install:
</br>
    1.install libxml2  
    </br>
    2.edit Makefile 
    </br>
        CINC = -I. -I changeHereToYourPath/libxml2     
        </br>
    3.make file and use
    </br>
usage: 
</br>
    ./xlsxTotxt file.xlsx 
</br>
output:
</br>
    sheet1.txt
